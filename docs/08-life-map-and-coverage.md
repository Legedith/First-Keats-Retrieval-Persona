# Life Map and Coverage

## Purpose

The life map helps the interviewer see what has been covered and what remains unexplored.

It is not a scorecard. It is not a claim that a person’s life can be completed.

Use it to avoid common blind spots.

## The life map

A simple life map places time periods against domains.

| Period | Place | Household | Work / school | Relationships | Public context | Notes |
|---|---|---|---|---|---|---|
| Childhood |  |  |  |  |  |  |
| Adolescence |  |  |  |  |  |  |
| Early adulthood |  |  |  |  |  |  |
| Middle years |  |  |  |  |  |  |
| Later years |  |  |  |  |  |  |

Better: use the narrator’s own chapter names.

Example:

| Narrator’s chapter | Approx. dates | Main places | Main people | Daily life | Turning points | Uncertainties |
|---|---:|---|---|---|---|---|
| “The hungry years” | ~1965–1972 | village, aunt’s house | mother, elder brother | farm work, school interruption | drought, father’s illness | dates uncertain |
| “Delhi beginning” | ~1978–1983 | rented room | coworkers, cousin | factory work | first salary, loneliness | exact move year uncertain |

## Coverage dimensions

For each chapter, check whether you have at least some material in these dimensions.

### 1. Scene

A bounded event with sequence.

Prompt:

> “Can you take me to one particular day?”

### 2. Ordinary life

Routine, not milestone.

Prompt:

> “What did a normal day look like?”

### 3. Relationships

People, power, care, conflict, absence.

Prompt:

> “Who mattered most in that period?”

### 4. Social context

Rules, expectations, constraints.

Prompt:

> “What would people around you have expected?”

### 5. Material context

Money, housing, food, transport, tools, labor.

Prompt:

> “What did daily life require physically?”

### 6. Choice and constraint

Agency within limits.

Prompt:

> “What choices seemed possible?”

### 7. Change

Before/after.

Prompt:

> “What changed because of this period?”

### 8. Meaning

Then and now.

Prompt:

> “How did you understand it then, and how do you see it now?”

### 9. Uncertainty

Memory status.

Prompt:

> “How certain do you feel about this part?”

### 10. Access

Sharing and privacy.

Prompt:

> “Who should be allowed to hear this?”

## Coverage sheet

Use a simple table.

| Chapter | Scene | Ordinary day | Relationships | Context | Choice / constraint | Meaning | Uncertainty | Restrictions | Follow-up |
|---|---|---|---|---|---|---|---|---|---|
|  | ☐ | ☐ | ☐ | ☐ | ☐ | ☐ | ☐ |  |  |

Use marks:

- `✓` covered.
- `~` partly covered.
- `?` uncertain.
- `X` narrator declined.
- `!` sensitive.
- `→` follow-up needed.

## Common blind spots

### Too much childhood, not enough later life

Many narrators remember youth vividly. Later adult years may be reduced to work and duty.

Ask:

> “What changed in you after 50?”  
> “What did later life teach you that youth did not?”

### Too many milestones, not enough routine

Ask:

> “What happened on an ordinary Tuesday?”

### Too much individual choice, not enough constraint

Ask:

> “What choices were actually available?”

### Too much public history, not enough personal impact

Ask:

> “How did that event enter your home?”

### Too much family tree, not enough relationship

Ask:

> “When did your relationship with them change?”

### Too much suffering, not enough humor

Ask:

> “Who made you laugh then?”

### Too much achievement, not enough invisible work

Ask:

> “What responsibility did you carry that others did not see?”

### Too much certainty

Ask:

> “Is this direct memory, family story, or a mixture?”

## Do not use a completion percentage

Never say:

> “We have captured 72% of your life.”

That is false and dehumanizing.

Say:

> “We have strong material on childhood and early work. We have less on friendship, ordinary domestic life, and your later years. Would you like to explore any of those?”

## Follow-up planning

After each session, choose no more than three follow-up priorities.

Good follow-up priority:

> “Ask about ordinary workday during factory years.”

Bad follow-up priority:

> “Finish marriage section.”

You are not completing forms. You are deepening a life narrative.
