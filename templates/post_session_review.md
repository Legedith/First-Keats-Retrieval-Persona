# Post-Session Review

Use with the narrator at the end of a session or at the start of the next one.

## Correction

Did I misunderstand anything?




Are there names, places, dates, or relationships to correct?




## Access

Is there anything from this session that should be:

- [ ] Open to agreed listeners
- [ ] Restricted
- [ ] Sealed until later
- [ ] Deleted or not used
- [ ] Discussed before sharing

Notes:




## Emotional check

How did this conversation feel afterward?




Would you like the next session to be:

- [ ] Lighter
- [ ] Similar depth
- [ ] More focused
- [ ] Shorter
- [ ] Longer
- [ ] With a different interviewer
- [ ] About a different topic

## Follow-up

What should we return to?




What should we not return to?




What question should I have asked?




## Narrator’s closing note

Is there anything you want to say directly to someone who may listen later?
